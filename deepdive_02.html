Deep Dive #2

Questions 1: What request type is common across all three (WMS, WFS, WCS) OGC web services that we have learned about?

GetCapabilities is common accross all three

Question 2a: What are the required parameters, and what do they represent?

  • SERVICE=WMS - Service type
  • REQUEST=GetCapabilities - Request Name

Question 2b What is returned in response to a WMS GetCapabilities request?

  • All interfaces a WMS service can support
  • Image formats it can serve (e.g. jpeg, png, gif)
  • List of spatial reference systems available for delivery of map data
  • List of all exception formats for return of exceptions
  • List of all vendor specific capabilities (or properties) that are available for modifying or controlling actions of the service with the current value of each capability
  • List of one or more map layers available in the service
  • List of layers supporting the optional GetFeatureInfo interface

Question 3

  • What are the required parameters, and what do they represent?
    • WMTVER - Request Version
    • Request=map - request name
    • Layers-layer_list - Map layers
    • Styles - Rendering style
    • SRS (depending on version) - Spatial reference system
    • CRS (depending on version) - Spatial reference system
    • BBOX - Bounding box corners in SRS units
    • WIDTH - Width in pixels of map picture
    • HEIGHT - Height in pixels of map picture
    • FORMAT - Output format of map
  • What is returned in response to a WMS GetMap request?
    • A map from a SWMS site
  • What is the significance of transparency in WMS requests?
    • Transparency is usefull for layering multiple WMS requests on top of each other.

Question 4: What OGC request would you use to inform the configuration of a client application (like ArcGIS) about an OGC service that you want to add layers from?

GetCapabilities

Question 5:Which OGC request would you submit under the following circumstances (include both the service type [e.g. WMS, WFS, WCS], and the request [e.g. GetMap, GetCapabilities, GetCoverage, etc.] in your answer)

  • You want a map image representing three layers of data in a single JPEG for a specified area of interest.
    • WMS GetMap
  • You want to retrieve data representing geometries and associated attributes for a road network, with the returned data in GML.
    • WFS GetFeature
  • You want to retrieve data representing a digital elevation model (a raster dataset) in the form of a GeoTIFF.
    • WCS GetCoverage
